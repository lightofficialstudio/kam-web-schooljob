# Landing Page Setup

The initial landing page has been created with a modern UI inspired by SchoolJob.

## File Location

- [app/pages/landing/page.tsx](app/pages/landing/page.tsx)

## Technologies Used

- **Ant Design**: For UI components (Layout, Card, Grid, Select, Button, Typography).
- **Tailwind CSS**: For custom styling and layout responsiveness.
- **Kanit Font**: Default typography.

## UI Components

- **Hero Section**: With a modern blue gradient background and rounded corners.
- **Dynamic Search Card**: Using Ant Design's Grid and Select components.
- **Feature Cards**: Interactive hoverable cards for key actions.
- **Floating Action Menu**: Replicating the sidebar functionality in a modern desktop/mobile floating UI.
