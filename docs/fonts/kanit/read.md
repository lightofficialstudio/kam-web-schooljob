# Font Configuration

This project uses **Kanit** as the default font.

## Setup

In `app/layout.tsx`:

- Import `Kanit` from `next/font/google`.
- Configure weights: `300, 400, 500, 600, 700`.
- Applied to body via `className` and CSS variable `--font-kanit`.

## Usage

Used via Tailwind class: `font-kanit`.
