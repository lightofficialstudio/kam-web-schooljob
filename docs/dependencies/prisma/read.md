# Prisma ORM

This project uses Prisma ORM for database management.

## Installation

Prisma has been installed as a dev dependency.

- `prisma`: Dev dependency
- `@prisma/client`: Production dependency

## Initialized Files

- `prisma/schema.prisma`: Schema definition file.
- `.env`: Database connection URL.
- `prisma.config.ts`: Prisma configuration.

## Common Commands

- `bunx prisma generate`: Generate the Prisma Client.
- `bunx prisma migrate dev --name init`: Create and apply a migration.
- `bunx prisma studio`: Open Prisma Studio.
